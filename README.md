# LA Fire Facts

A simple, responsive website that shares key facts about the 2025 LA wildfires.

## Features

- Sticky navbar
- Grid layout with fire facts
- Banner with overlay text
- Responsive design for mobile
- Footer with social links

## Built With

- HTML
- CSS (Flexbox & Grid)
- Google Fonts
- Font Awesome icons
